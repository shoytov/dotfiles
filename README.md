# TMUX

Install tmux on operational system like that:

```shell
brew install tmux
```

```shell
sudo pacman -S tmux

```

etc.

## Config

```
cd
git clone --single-branch https://github.com/gpakosz/.tmux.git
ln -s -f .tmux/.tmux.conf
cp ~/dotfiles/.tmux.conf.local ~/.tmux.conf.local
```
